Composer для автозагрузки:

	создаем composer.json в корне проекта. Прописываем :

{
  "autoload": {
    "psr-4": {
      "namespace\\example1\\": "app",
      "namespace\\example2\\": "app1"
    }
  }
}

namespace\\example1\\ - название пространства имен
app - папка, в которой будут находится файлы пространства имен

для применения настроек используем composer dump-autoload и подключаем autoload.php к нашему файлу с приложением

__DIR__ - это буть до файла в котором эта директива используется
